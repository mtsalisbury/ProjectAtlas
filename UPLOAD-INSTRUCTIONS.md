# Upload instructions

1. Open the clean local GitHub Desktop repository folder named `ProjectAtlas`.
2. Copy everything from this package into that folder:
   - `README.md`
   - `UPLOAD-INSTRUCTIONS.md`
   - `docs/`
   - `source/`
   - `archive/`
3. In GitHub Desktop, confirm the changed files are listed.
4. Enter the commit summary: `Project Atlas clean baseline`.
5. Click **Commit to main**.
6. Click **Push origin**.
7. On GitHub, open **Settings → Pages**.
8. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
9. Save and wait for the Pages deployment.
10. Open `https://mtsalisbury.github.io/ProjectAtlas/`.
