# Project Atlas

Project Atlas is an open research initiative exploring a person-controlled Internet presence that can persist across networks, providers, devices, applications, and institutions.

## Website structure

The public website is published directly from the `docs/` folder on the `main` branch.

- `docs/index.html` — homepage
- `docs/style.css` — homepage stylesheet
- `docs/research/index.html` — research portal
- `docs/research/papers/` — published Atlas Papers
- `source/papers/` — editable research source files
- `archive/` — preserved pre-reset repository backup

## Publishing

GitHub Pages configuration:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/docs**

No GitHub Actions workflow, MkDocs build, or `gh-pages` branch is required.
